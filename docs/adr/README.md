# Architecture Decision Records (ADRs)

This directory contains the Architecture Decision Records for the PitWall AI project.

## What is an ADR?

An Architecture Decision Record (ADR) is a short text file in a format similar to an Alexandrian pattern that describes a set of forces and a single decision in response to those forces.

## How to add an ADR

1. Copy `docs/templates/adr-template.md` to a new file in this directory.
2. Name the file sequentially (e.g., `0001-record-architecture-decisions.md`).
3. Fill out the template.
4. Submit a Pull Request for review.
